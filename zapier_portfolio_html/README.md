# Zapier Operations Automation Portfolio

Prepared by: Jhun Bart Macirin

## How to Use

1. Open `index.html` in your browser.
2. Edit the text in `index.html` if you want to customize the portfolio.
3. Replace placeholder images inside the `images` folder with your actual screenshots.
4. Update the portfolio links in the `Portfolio Links` section.
5. You may upload this folder to GitHub Pages, Netlify, Vercel, or any static hosting platform.

## Suggested Screenshot File Names

You can save your screenshots using names like:

- client-inquiry-zap.png
- client-inquiry-form.png
- client-inquiry-tracker.png
- task-request-zap.png
- task-request-card.png
- client-onboarding-zap.png
- onboarding-drive-folder.png

Then update the image source in `index.html`.

Example:

```html
<img src="images/client-inquiry-zap.png" alt="Client Inquiry Zapier Screenshot" />
```
